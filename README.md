# RCMS-GA (Radio Control and Monitoring System for General Aviation)

Универсальная система управления радиостанциями гражданской авиации РФ по RS-485.

## Цель проекта

Создание профессионального, сертифицируемого ПО для дистанционного управления и мониторинга радиопередатчиков/радиоприемников гражданской авиации РФ.

## Почему это нужно

- **Завод "Электроприбор" не предоставляет ПО** — только протокол в РЭ
- **Существуют только любительские самоделки** без поддержки и гарантий
- **Нет единого решения** для разных производителей оборудования
- **Потребность в Linux-решении** для госструктур (Astra, РЕД ОС)
- **Потенциальный заказчик** — Госкорпорация по ОрВД

## Технологии

- **Язык**: C++17
- **GUI**: Qt 5.15 / Qt 6
- **Платформа**: Linux (Debian, РЕД ОС)
- **Протокол**: Modbus RTU по RS-485

## Поддерживаемое оборудование

### Приоритет 1: Фазан-19 (Электроприбор Владимир)
- Фазан-19 Р5 (5 Вт)
- Фазан-19 Р50 (50 Вт)

### Приоритет 2 (в перспективе)
- РС П1000
- Оборудование АО "Азимут" (TRS 2000)

## Функционал

- Мониторинг состояния радиостанций
- Управление частотой
- Управление подавителем шума (ПШ)
- Управление передачей (PTT)
- Диагностика неисправностей
- Журналирование событий
- Поддержка нескольких устройств на шине RS-485
